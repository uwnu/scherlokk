## 为什么 Spotlight 总是让人失望？

更让人头疼的是，Spotlight 的搜索逻辑有时候实在让人摸不着头脑。你明明输入了准确的文件名，它却给你返回一堆不相干的结果；想搜索文档里的某个关键词，结果发现根本搜不出来。这种时候，你就会意识到，原来 Spotlight 只是个"够用"的工具，而不是"好用"的工具。
